# FarmacIA — Demo

Mockup navegável da plataforma de pré-atendimento farmacêutico com IA.

🔗 **Ver demo:** https://matheusdexty.github.io/farmacia-ia-demo/

## Sobre

Esse é um repositório público apenas com o mockup demonstrativo. O produto, documentação completa e código de produção ficam em repositório privado.

A demo cobre 5 telas:
1. Boas-vindas e consentimento LGPD
2. Anamnese conversacional (chatbot guiado pela IA)
3. Ficha gerada com sugestão preliminar de princípio ativo
4. Cenário de red flag (encaminhamento médico)
5. Painel do farmacêutico com fila de casos clicáveis e tomada de decisão

## Princípios do produto

- Farmacêutico no loop sempre — IA não decide sozinha
- Red flags determinísticas (não dependem do LLM)
- A IA pode dizer "não sei"
- Auditabilidade total
- Privacidade primeiro (LGPD Art. 11)

## Stack do mockup

HTML + Tailwind CSS via CDN. Single-page, sem build, sem dependências. Abre em qualquer navegador moderno.
